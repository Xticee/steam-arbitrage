# Steam Arbitrage

Сервис для арбитража скинов Rust — покупаешь дешевле на lis-skins.com, продаёшь дороже на Steam Market.

## Как это работает

1. Парсер забирает цены с [lis-skins.com](https://lis-skins.com) каждые 10 минут
2. Параллельно непрерывно парсит цены Steam Market (~5120 предметов)
3. Считает профит с учётом комиссии Steam (13%) и маркета (5.821%)
4. Веб-интерфейс показывает топ предметов для заданной суммы пополнения

## Структура

```
├── cmd/
│   └── main.go
├── internal/
│   ├── market/     # парсинг lis-skins.com
│   ├── steam/      # парсинг Steam Market
│   ├── db/         # PostgreSQL
│   ├── updater/    # фоновые горутины обновления цен
│   └── api/        # chi роутер + фронтенд
├── .env
└── go.mod
```

## Стек

- **Go** — бэкенд
- **PostgreSQL** — хранение цен
- **chi** — HTTP роутер
- **lib/pq** — драйвер PostgreSQL
